# Per-Job Tailoring Workflow

For the ~20% of jobs you genuinely want, spend 10 minutes tailoring instead of
auto-applying. Tailored applications convert to interviews several times more
often than generic ones. This is the highest-ROI hour of your job search.

## The 10-minute tailoring loop

### 1. Extract keywords from the job description (3 min)
Copy the posting. Pull out:
- **Hard skills** named (e.g., "PyTorch", "feature stores", "A/B testing", "Spark").
- **Repeated phrases** — anything mentioned 2+ times is what the ATS weights.
- **Must-haves** in the requirements section.

### 2. Mirror them in your resume (4 min)
- Add any *true* matching skill to your Skills line using the **exact wording** of
  the posting (ATS matches strings: "ML" ≠ "Machine Learning" to a dumb parser —
  include both forms if the JD uses both).
- Edit your **Summary** first line to echo the role title and top requirement.
- Reorder Experience bullets so the most relevant one is first under each job.
- **Never add a skill you don't have.** Gaps get exposed in interviews.

### 3. Write a 4-sentence cover letter (3 min)
Use the master from `screening-answers.md §C`, swapping only sentence 2:
> *Your work on [specific product/team/problem from the JD] maps directly to my
> experience [specific true overlap].*

## ATS keyword-gap self-check

Paste your resume and the JD side by side. For each must-have requirement:

| JD requirement | On my resume? | Action |
|---|---|---|
| e.g. "production ML in Python" | ✅ | Keep, make it bullet #1 |
| e.g. "feature stores" | ❌ but I've used similar | Add honestly if true, else leave |
| e.g. "5+ yrs experience" | Borderline | Make total years explicit in summary |

If you're missing **most** must-haves, skip the job — that's a targeting signal,
not a tailoring problem.

## Cover letter template (copy/paste)

```
Dear Hiring Team,

I'm a Machine Learning Engineer with production experience at Meta and an MS from
Imperial College London. Your work on [SPECIFIC PRODUCT/PROBLEM FROM JD] maps
directly to my experience [SPECIFIC TRUE OVERLAP]. I've [ONE QUANTIFIED
ACHIEVEMENT, e.g. "shipped a recommendation model improving engagement X% for N
users"], and I'm confident I can deliver similar impact for [COMPANY].

I'd welcome the chance to discuss how I can contribute to your team.

Best regards,
[Your Name]
```
